# Inputs and Outputs

Write down pseudo-code on pen and paper for the following patterns:

1.  Input => Output
    2, 5 => 5
    0, 100 => 100
    52, 35 => 52

---

If user inputs 2
Print "5"

If user inputs 0
Print 100

If user inputs 35
Print 52

---

2.  Input => Output
    5, 5 => true
    40, 40 => true
    50, 51 => false

---

If 5 === 5
Print "true"

If 40 === 40
Print "true"

If 50 < 51
Print "false"

---

3. Input => Output
   110, 100 => true
   150, 100 => true
   50, 100 => false

---

If 100 < 110
Print "true"

If 100 < 150
Print "true"

If 100 < 50
Print "false"
